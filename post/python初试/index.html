Python初试

2019-05-28

最近在做一个关于机器人的项目，需要接收和分析卫星定位数据，并把位置显示到地图上。初次尝试，只用python写了一个简单的读取脚本，记录一下注意事项。

前期学了几天python，也简单了解了NEMA 0183协议和GNSS接收机的命令和数据收取。 经过在网上学习了很多别人的代码后，我完成了以下脚本：

#设置脚本编码
#coding=utf-8

#导入必要的模块，这里的pynmea2暂时没有用到，下面解释它的用处
import serial
import pynmea2
import time

#配置并打开串口，这里是在Windows下的串口
ser = serial.Serial("COM4",115200)
ser.timeout = 2 #设置超时时间为两秒
ser.parity = "N" #设置校验码为无

#检测串口是否打开
def Open():
    if ser.isOpen():
        print("serial is opened.")
    else:
        ser.open()
        print('serial is restarted.')

				#发送命令
def SendDate(i_msg):
    i_msg = i_msg + "\r\n" #要加上换行回车符表示命令传输完成
    try:
        i_msg = i_msg.encode('ascii') #将string字符串转换为byte字节
        ser.write(i_msg) #发送命令
    except Exception as ex:
        print(ex)

#开始执行
if __name__ == "__main__":
    Open()
    SendDate("KSXT 1")
    while 1:
        line = ser.readline() #读取数据，这里是按行读取，遇到"\n"结束
        line = line.decode('ascii') #将读取出来的byte型转为string型
        print(line)

注意事项：

  1. 如果用的readline读取一定要设置timeout，否则可能造成阻塞
  2. 注意发送数据一定要转换成字符型，否则会直接报错
  3. 命令要加上"\r\n"作为结束标志
  4. serial读出来的数据为byte型，要通过decode方法解码成string型，编码按接收的数据来设置

  2. 利用ROS在Ubuntu上读取数据，实现以上筛选。
  3. 将读取的位置信息显示在地图上（可以利用各大地图的API实现）。
